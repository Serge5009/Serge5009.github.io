Empire Grinder
All images

Description:

Empire Grinder is a raw but interesting variant of "clicker".
This game is inspired by "Idle" game genre, where you spend resources to increase your income.
This game was created in "MitApp Inventor", this tool isn't powerful enough to crate normal game, but it's faster way of creating templates like this one.
The music in this project is made by S&T. I've made volume controls, language settings and some other features.
