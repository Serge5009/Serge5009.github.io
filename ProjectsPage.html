Jenga Unity -- (in development)
All images

Description:

Simple unity physics based Jenga. It has no UI, so you have to restart it using Alt+F4. I made it just for fun, and it was fun for first 15 minutes.
There are also some other objects on the table that can fall as you destroy your tower.
At this moment game is unbalanced and it's hard to move more than 4 blocks without breaking the tower.

Empire Grinder
All images

Description:

Empire Grinder is a raw but interesting variant of "clicker".
This game is inspired by "Idle" game genre, where you spend resources to increase your income.
This game was created in "MitApp Inventor", this tool isn't powerful enough to crate normal game, but it's faster way of creating templates like this one.
The music in this project is made by S&T. I've made volume controls, language settings and some other features.
